FacebookRestClient::application_getPublicInfo

Returns public information for an application (as shown in the application directory) by either application ID, API key, or canvas page name.

    Signature

    public function application_getPublicInfo([int $application_id = null, [string $application_api_key = null, [string $application_canvas_name = null]]] )

    Parameters

    $application_id — int
    (Optional) app id
    $application_api_key — string
    (Optional) api key
    $application_canvas_name — string
    (Optional) canvas name
    Exactly one argument must be specified, otherwise it is an error.

    Returns

    array
    An array of public information about the application.
