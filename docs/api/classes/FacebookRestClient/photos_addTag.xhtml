FacebookRestClient::photos_addTag

Adds a tag with the given information to a photo. See the wiki for details:

http://wiki.developers.facebook.com/index.php/Photos.addTag

    Signature

    public function photos_addTag(int $pid, [int $tag_uid = 0, [string $tag_text = 0, [float $x = 0, [float $y = 0, [array $tags = 0, [integer $owner_uid = 0]]]]]] )

    Parameters

    $pid — int
    The ID of the photo to be tagged
    $tag_uid — int
    The ID of the user being tagged. You must specify
    either the $tag_uid or the $tag_text parameter
    (unless $tags is specified).
    $tag_text — string
    Some text identifying the person being tagged.
    You must specify either the $tag_uid or $tag_text
    parameter (unless $tags is specified).
    $x — float
    The horizontal position of the tag, as a
    percentage from 0 to 100, from the left of the
    photo.
    $y — float
    The vertical position of the tag, as a percentage
    from 0 to 100, from the top of the photo.
    $tags — array
    (Optional) An array of maps, where each map
    can contain the tag_uid, tag_text, x, and y
    parameters defined above. If specified, the
    individual arguments are ignored.
    $owner_uid — integer
    (Optional) The user ID of the user whose photo
    you are tagging. If this parameter is not
    specified, then it defaults to the session user.

    Returns

    bool
    true on success
