FacebookRestClient::profile_setInfo

Configures an application info section that the specified user can install on the Info tab of her profile. For details on the structure of an info field, please see:

http://wiki.developers.facebook.com/index.php/Profile.setInfo

    Signature

    public function profile_setInfo(string $title, [int $type = null, [array $info_fields = null, [int $uid = null]]] )

    Parameters

    $title — string
    Title / header of the info section
    $type — int
    1 for text-only, 5 for thumbnail views
    $info_fields — array
    An array of info fields. See wiki for details.
    $uid — int
    (Optional)

    Returns

    bool
    true on success
