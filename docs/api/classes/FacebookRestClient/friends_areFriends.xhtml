FacebookRestClient::friends_areFriends

Returns whether or not pairs of users are friends.

Note that the Facebook friend relationship is symmetric.

    Signature

    public function friends_areFriends(array/string $uids1, array/string $uids2 )

    Parameters

    $uids1 — object
    list of ids (id_1, id_2,...)
    of some length X (csv is deprecated)
    $uids2 — object
    list of ids (id_A, id_B,...)
    of SAME length X (csv is deprecated)

    Returns

    array
    An array with uid1, uid2, and bool if friends, e.g.:
    array(0 => array('uid1' => id_1, 'uid2' => id_A, 'are_friends' => 1), 1 => array('uid1' => id_2, 'uid2' => id_B, 'are_friends' => 0) ...)
