FacebookRestClient::fql_multiquery

Makes a set of FQL queries in parallel. This method takes a dictionary of FQL queries where the keys are names for the queries. Results from one query can be used within another query to fetch additional data. More info about FQL queries at http://wiki.developers.facebook.com/index.php/FQL

    Signature

    public function fql_multiquery(string $queries )

    Parameters

    $queries — string
    JSON-encoded dictionary of queries to evaluate

    Returns

    array
    generalized array representing the results
