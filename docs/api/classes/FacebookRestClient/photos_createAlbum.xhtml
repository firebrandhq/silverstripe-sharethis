FacebookRestClient::photos_createAlbum

Creates and returns a new album owned by the specified user or the current session user.

    Signature

    public function photos_createAlbum(string $name, [string $description = '', [string $location = '', [string $visible = '', [integer $uid = 0]]]] )

    Parameters

    $name — string
    The name of the album.
    $description — string
    (Optional) A description of the album.
    $location — string
    (Optional) A description of the location.
    $visible — string
    (Optional) A privacy setting for the album.
    One of 'friends', 'friends-of-friends',
    'networks', or 'everyone'. Default 'everyone'.
    $uid — integer
    (Optional) User id for creating the album; if
    not specified, the session user is used.

    Returns

    array
    An album object
