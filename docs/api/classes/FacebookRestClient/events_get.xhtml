FacebookRestClient::events_get

Returns events according to the filters specified.

    Signature

    public function events_get([int $uid = null, [array/string $eids = null, [int $start_time = null, [int $end_time = null, [string $rsvp_status = null]]]]] )

    Parameters

    $uid — int
    (Optional) User associated with events. A null
    parameter will default to the session user.
    $eids — object
    (Optional) Filter by these event
    ids. A null parameter will get all events for
    the user. (A csv list will work but is deprecated)
    $start_time — int
    (Optional) Filter with this unix time as lower
    bound. A null or zero parameter indicates no
    lower bound.
    $end_time — int
    (Optional) Filter with this UTC as upper bound.
    A null or zero parameter indicates no upper
    bound.
    $rsvp_status — string
    (Optional) Only show events where the given uid
    has this rsvp status. This only works if you
    have specified a value for $uid. Values are as
    in events.getMembers. Null indicates to ignore
    rsvp status when filtering.

    Returns

    array
    The events matching the query.
