FacebookRestClient::photos_get

Returns photos according to the filters specified.

    Signature

    public function photos_get(int $subj_id, int $aid, array/string $pids )

    Parameters

    $subj_id — int
    (Optional) Filter by uid of user tagged in the photos.
    $aid — int
    (Optional) Filter by an album, as returned by
    photos_getAlbums.
    $pids — object
    (Optional) Restrict to an array of pids
    (csv is deprecated)

    Note that at least one of these parameters needs to be specified, or an
    error is returned.

    Returns

    array
    An array of photo objects.
