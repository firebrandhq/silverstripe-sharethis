FacebookRestClient::profile_setInfoOptions

Specifies the objects for a field for an application info section. These options populate the typeahead for a thumbnail.

    Signature

    public function profile_setInfoOptions(string $field, array $options )

    Parameters

    $field — string
    The title of the field
    $options — array
    An array of items for a thumbnail, including
    'label', 'link', and optionally 'image',
    'description' and 'sublabel'

    Returns

    bool
    true on success
