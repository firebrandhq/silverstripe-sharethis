FacebookRestClient::feed_publishUserAction

Publishes a story on behalf of the user owning the session, using the specified template bundle. This method requires an active session key in order to be called.

The parameters to this method ($templata_data in particular) are somewhat involved. It's recommended you visit the wiki for details: http://wiki.developers.facebook.com/index.php/Feed.publishUserAction
