FacebookRestClient::fbml_registerCustomTags

Register custom tags for the application. Custom tags can be used to extend the set of tags available to applications in FBML markup.

Before you call this function, make sure you read the full documentation at http://wiki.developers.facebook.com/index.php/Fbml.RegisterCustomTags IMPORTANT: This function overwrites the values of existing tags if the names match. Use this function with care because it may break the FBML of any application that is using the existing version of the tags.
